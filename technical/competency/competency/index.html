    COMPETENCY

    DEVOPS ROADMAP

    CHECKLIST

    Kiến trúc máy tính và Hệ điều hành

    • Hiểu về chức năng, nguyên tắc hoạt động của CPU, GPU, RAM, HDD, IO và cách thức chúng tương tác với nhau.
    • Hiểu biết về vai trò của hệ điều hành và cơ chế HĐH điều khiển các thành phần của máy tính.
    • Nắm rõ về các cơ chế quản lý process, file, storage, IO, multi-tasking.

    Mạng máy tính

    • Hiểu biết mô hình OSI, TPC/IP
    • Hiểu biết về Ethernet (FE, GE, 10GE).
    • Hiểu biết về IP addressing, IP subneting, private vs. public IP address,
    • Hiểu rõ về VLAN, VLAN tagging and VLAN trunking.
    • Hiểu được vai trò của các thiết bị mạng phổ biến như switch, router, firewall.

    An ninh hệ thống máy tính

    • Hiểu và phân loại các mối đe doạ (threats) đến an ninh máy tính.
    • Hiểu biết các phương thức tấn công phổ biến trên mạng
    • Phương pháp bảo vệ hệ thống máy tính bằng các phương pháp hardenning OS và application

    Protocol

    • DHCP

    • Telnet

    • VPN
    • HTTP
    • ...

    OS Linux

    • Hiểu biết cơ chế quản lý process/ thread trên linux
    • Hệ thống file system (fat, ntfs, ext3, xfs...),
    • Cơ chế quản lý IO với NIC và RAID cards.
    • Hiểu rõ cơ chế quản lý init scripts/services, package/repo.
    • Hiểu biết thành thạo các biện pháp để hardenning linux box (tuning, tắt các dịch vụ không cần thiết, recompile kernel & đóng gói).
    • Hiểu rõ các phương pháp monitor và tracing events trong linux box như tcpdump hoặc syslog.

    Webserver/ Proxy

    • Nginx
    • Apache
    • Haproxy
    • Squid

    • Mongodb
    • Postgres
    • Couchbase
    • ...

    Plus

    • Quản lý performance và tuning kernel OS để đạt performance tốt nhất
    • Hiểu biết về Hadoop và ứng dụng của Hadoop trong data processing.
    • Hiểu biết chuyên sâu các giao thức kết nối mạng và storage: TCP/IP, IP Routing, NFS, SCSI, iSCSI, FCoE... hiểu biết về các cơ chế link aggregation (bonding) để cung cấp khả năng load balancing và HA cho hệ thống.
    • Có kiến thức căn bản về system virtualization, nguyên tắc hoạt động, các thành phần cần thiết và ứng dụng của mỗi mô hình virtualization.
